# MAC Serviços - Gestão de Ordens de Serviço

Sistema de gestão mobile-first para prestadores de serviços, focado em facilidade de uso e relatórios rápidos via WhatsApp.

## 🚀 Como colocar online (Vercel)

Para deixar seu aplicativo acessível de qualquer lugar, siga estes passos:

1. **GitHub**:
   - Crie um novo repositório no seu GitHub.
   - Suba todos os arquivos desta pasta para lá.

2. **Vercel**:
   - Acesse [vercel.com](https://vercel.com) e faça login com seu GitHub.
   - Clique em **"Add New"** > **"Project"**.
   - Importe o repositório que você acabou de criar.
   - Clique em **"Deploy"**.

Pronto! A Vercel vai te dar um link (ex: `mac-servicos.vercel.app`) e seu sistema estará online.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Lucide Icons
- LocalStorage para persistência de dados (os dados ficam salvos no navegador do celular que você usar).

## 🔒 Segurança de Dados
Os dados são salvos localmente no navegador. Lembre-se de usar a função de **Exportar Backup** na aba de Configurações regularmente para não perder seus dados caso troque de celular ou limpe o cache.
